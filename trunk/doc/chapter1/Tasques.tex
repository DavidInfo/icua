\section{Tasques a realitzar}
Durant la realització del projecte S'han hagut de realitzar les següents tasques que considero claus,

 \begin{list}{-}{}


 \item Instalació de l'entorn de desenvolupament en el que es treballará.
 
 \subitem * S.O. Debian 4.
 \subitem * Elcipse + Android Plugins.
 \subitem * Java Runtime Environment 1.5.
 \subitem * SDK d'Android.
 \subitem * Compilador de \LaTeX.
 

 \item Lectura dels meta tags dels MP3. Tots els MP3 tenen la possibilitat d'emagatzemar dades sobre la cançó, com per exemple el títol, l'artista, l'album, la portadada del disc, etc... tot aixó mitjançant el format estandard ID3.
 
 
 \end{list}
