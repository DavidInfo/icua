\subsection{Argo UML}
ArgoUML es l'aplicació de software lliure escollida per a la realització dels casos d'ús, models conceptuals i diagrames de seqüència de la aplicació a desenvolupar.

ArgoUML está creat amb java, i necesitarém alguna Java Runtime Enviroment per poder llançar-lo. la versió a utilitzar es la 0.26.2

\subsubsection{Llicència}
Argo UML esta sota llicència de software lliure BSD i funcióna sobre qualsevol plataforma amb suport per Java.
